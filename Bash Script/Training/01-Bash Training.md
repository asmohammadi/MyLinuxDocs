## Making the file executable
- The command **chmod** A.K.A change mode is used to change the access permissions of a file 
- you can check the access permissions using ```ls -l ```
```shell
chmod +x ./challenge_one.sh
```

## Running the script 
- /path/to/the/script

```shell
./challenge_one.sh
```
